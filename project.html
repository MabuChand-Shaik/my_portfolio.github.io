Data Integrity Auditing Without Private Key Storge for Secure Cloud Storage

   Designed a practical data integrity auditing scheme without private key storage for secure cloud storage. In this scheme, two fuzzy private keys(biometric data) are extracted from the user in the phase of registration and the phase of signature generation.

   Designed a new signature scheme names as MBLSS by modifying the BLS short signature based on the idea of fuzzy signature. Here, the security analysis and justify the performance via concrete implementations. The results show that the proposed scheme is secure and efficient.
